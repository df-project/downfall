Mandelbrot

  • Mandelbrot definition :
  • $$z_0 = 0$$
  • $$z_{n+1} = z_n^2 + c$$ with $$z = x + iy$$
  • Representation :
  • - If $$|z_i| \geq 2$$ : escapes to infinity and point with color $$i$$
  • - If $$|z_i| < 2$$ : belongs to the set and point in black
